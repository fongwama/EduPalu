EduPalu est une application d'information et d'éducation sur le paludisme au Congo.

Cette application utilise les technologies web les plus récentes : HTML5, CSS3 et JavaScript.

EduPalu est un logiciel libre, que vous pouvez utiliser, modifier, copier comme bon vous semble.
Le code source d'EduPalu est disponible sur le site de développement collaboratif GitHub. Vous pouvez le télécharger à cette adresse :
http://github.com/fongwama/EduPalu
